# Instapro

MVP социальной сети для обмена фотографиями

## Первоначальная оценка

40 часов

## Фактически затраченное время

400 часов

## Функциональность

- возможность ставить лайки / дизлайки;
- открывать страницы всех постов и отдельного пользователя;
- создание поста (загрузка фотографий с описанием).
